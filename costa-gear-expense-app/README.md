# Costa Gear Business Expenses Tracker

React app for managing business expenses, asset purchases and estimated CCA for tax reporting.

## Commands

```bash
npm install
npm start
npm run build
```

## Required environment variables

```txt
REACT_APP_SUPABASE_URL=
REACT_APP_SUPABASE_ANON_KEY=
```

## Main files

```txt
src/App.js
src/styles.css
src/supabase.js
supabase-setup.sql
DEPLOY-GUIDE.md
public/costa-gear-logo.png
```
